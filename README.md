# Instalily Case Study Backend

Forked from starter kit for developing backend with Node.js

Runs **locally** on port 3000

### Run

```bash
npm start
```

### Run development

```bash
npm run dev
```

### Linter

```bash
npm run lint
```

```bash
npm run lint:fix
```

### Docker

```bash
docker build --tag sskender/app:latest .
```
